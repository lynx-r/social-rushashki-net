Добро пожаловать на страницу анонса социальной сети для шашистов.

Здесь я поделюсь своими планами о том, что будет сделано.

Я хочу сделать социальные шашки. Как твиттер, только посты - это игры. С комментариями, лайками, хеш-тегами и чатом.

Так же, я хочу распознавать шашечные дебюты и выводить их название рядом с аннотацией. Например: "Вы разыграли дебют Городская партия". Или: "Вы разыграли дебют Защита Соков". И сделать ещё несколько интеллектуально-обучающих фич, которые я ещё не придумал :)

Кроме этого, я собираюсь произвести ряд улучшений по сравнению с предыдущей версией сайта http://rushashki.net.

И наконец, если вы знакомы с такими технологиями как Java 8 EE, WildFly 8, GWT 2.8 и Git, то предлагаю на безвозмездной основе внести вклад в развитие проекта. Форкайте репозиторий и делайте пул реквесты :)

$ git clone https://github.com/lynx-r/social-rushashki-net.git
$ cd social-rushashki-net
$ git checkout develop
